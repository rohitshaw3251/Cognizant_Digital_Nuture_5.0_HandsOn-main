# Cognizant Digital Nurture 5.0 Hands-On Exercises

This repository contains hands-on training exercises for Java Full Stack Engineering (FSE) Deepskilling.

---

## Directory Structure

### [Week 1](file:///e:/Projects/Java FSE/week1/)
Contains exercises on foundational programming concepts:
- **Design Patterns and Principles**: Builder, Adapter, Decorator, Proxy, Observer, Strategy, Command, MVC, and Dependency Injection examples.
- **Data Structures and Algorithms**: Search mechanisms, sorting customer orders, custom singly linked lists for task scheduling, and library searches.
- **PL/SQL Programming**: Database schema setups, error handlers, user-defined functions, custom triggers, cursors, and management packages.
- **JUnit and Mockito**: Basic and advanced testing (parameterized inputs, category tags, expected exceptions, mock databases/APIs/file I/O).
- **SLF4J Logging**: Parameterized logging placeholder formats and custom appender simulators.

### [Week 2](file:///e:/Projects/Java FSE/week2/)
Contains exercises on enterprise application frameworks:
- **Spring Core & Maven**: Setter/Constructor dependency injection using both XML configuration files and component annotations.
- **Spring AOP**: LoggingAspect tracking method performance.
- **Spring Data JPA**: Relational database operations, derived search methods, custom JPQL/native SQL finders, transaction auditing, and page requests.

### [Week 3](file:///e:/Projects/Java FSE/week3/)
Contains exercises on web services and security:
- **Spring REST**: Building REST APIs with proper resource naming conventions, request validations, and global controller advice exception handling.
- **Spring Security & JWT**: Generating and parsing JSON Web Tokens (JWT) for secure REST resource requests.
